This is a simple mechanism for recording timestamped messages. Add a message by saying "confession: <whatever>" in a Matrix channel that contains the 'botzilla' bot. All messages are stored in https://github.com/robotzilla/histoire. Note that this message store, and the whole mechanism, is 100% completely public.

If you wish to modify any existing messages, you can submit a PR to the github repository. You will also need to ping @bnjbvr on Github to have updates merged. The matrix bot uses a special append-only merge driver, so hopefully will not get conflicts unless you delete files or something.

Feel free to file issues on the github repo; robotzilla may or may not ignore them.
