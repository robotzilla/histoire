This is a simple mechanism for recording timestamped messages. Add a message by saying "confession: <whatever>" in a Matrix channel that contains the 'botzilla' bot. All messages are stored in https://github.com/robotzilla/histoire. Note that this message store, and the whole mechanism, is 100% completely public.

If you wish to modify any existing messages, you can submit a PR to the github repository. Currently, you will also need to ping bnjbvr:matrix.org on Matrix or email ben@mozilla.com to have updates merged, but I plan to enable one of the many auto-merge bots to just blindly accept anything (once I figure out which one to use). The matrix bot uses a special append-only merge driver, so hopefully will not get conflicts unless you delete files or something.

Feel free to file issues on the github repo; robotzilla may or may not ignore them.
